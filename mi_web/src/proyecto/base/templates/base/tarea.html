Título del Contenedor

Ejemplo de un contenedor borroso. El contenido detrás del contenedor se verá desenfocado, creando un efecto de vidrio esmerilado.

/* Cambiar colores de enlaces */ a { color: #00ff00; /* Color del enlace normal */ text-decoration: none; /* Sin subrayado */ }

a:hover { color: #ff0000; /* Color al pasar el mouse sobre el enlace */ text-decoration: underline; /* Subrayado al pasar el mouse */ }

a:visited { color: #8888ff; /* Color del enlace visitado */ }

a:active { color: #ffff00; /* Color del enlace cuando es clicado */ }
